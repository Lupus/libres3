LibreS3 - Amazon S3 compatible server

What is LibreS3?

LibreS3 is a high performance daemon which makes SX Cluster fully compatible with AWS S3 APIs. With LibreS3 you access the data stored in your storage cluster using any existing S3 client. It is fully compatible with Amazon S3 API, a drop-in replacement for your expensive AWS S3 account. Deploy your own S3 cloud today and stop paying high bills for your S3 network transfer. Enjoy total privacy and faster transfer rates.

FAQ

Features

S3 Compatibility

LibreS3 is fully S3 compatible. Use it as a drop-in replacement for AWS S3.

Scalable

Install multiple LibreS3 nodes and add all of them to a round-robin to load balance the traffic among them.

Public buckets

LibreS3 has support for making buckets public and serving them over regular http / https. Use it to build your own CDN / CloudFront alternative!
